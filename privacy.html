🔭 SCOPE

Política de Privacidad

Última actualización: junio de 2026

Resumen en una línea: Scope no recopila, almacena ni vende tu historial de navegación. Nunca. Es exactamente lo contrario de lo que hace WOT.

1. Quiénes somos

Scope es una extensión de Chrome desarrollada de forma independiente. Puedes contactarnos en hola@getscope.app.

2. Qué datos NO recopilamos

  • Tu historial de navegación
  • Las URLs de los sitios que visitas (más allá de lo descrito abajo)
  • Datos personales de ningún tipo
  • Dirección IP, dispositivo, o información de identificación

3. Qué hace Scope con los datos

Detección de trackers (completamente local): Scope compara los dominios de terceros cargados en la página contra una base de datos incluida en la extensión. Esta comparación ocurre únicamente en tu dispositivo. Ningún dato sale de tu navegador para este proceso.

Consultas de seguridad (por sitio visitado): Cuando visitas un sitio web, Scope envía la URL de ese sitio a nuestro servidor intermedio (Cloudflare Worker), que a su vez consulta:

  • Google Safe Browsing API — para comprobar si el sitio está en listas de phishing o malware
  • VirusTotal API — para comprobar si motores antivirus lo marcan como peligroso

Estas consultas son únicamente sobre el sitio que estás visitando, no sobre ti. Nuestro servidor no registra estas consultas ni las asocia con ningún usuario.

Antigüedad del dominio: Scope consulta el servicio público RDAP de ICANN (rdap.org) para obtener la fecha de registro del dominio. Este servicio es público y no requiere identificación.

4. Almacenamiento local

Scope utiliza chrome.storage.local para almacenar en tu propio navegador:

  • El número de veces que has abierto el popup (para el prompt de reseña)
  • Tu clave de licencia Pro (si la tienes), para no validarla en cada visita

Estos datos nunca se envían a ningún servidor. Puedes eliminarlos en cualquier momento desinstalando la extensión.

5. Licencias Pro

Si adquieres Scope Pro, el proceso de pago lo gestiona Lemon Squeezy, un proveedor de pagos externo. Consulta su política de privacidad para entender cómo tratan tus datos de pago. Nosotros solo recibimos la confirmación de que la licencia es válida.

6. Servicios de terceros

La extensión interactúa con los siguientes servicios externos, exclusivamente para analizar el sitio que estás visitando:

  • Google Safe Browsing API — Política de Google
  • VirusTotal API — Política de VirusTotal
  • RDAP.org — Servicio público de ICANN, sin política de privacidad específica (servicio de consulta pública)
  • Cloudflare Workers — Nuestro servidor intermediario, sin logging de peticiones habilitado. Política de Cloudflare

7. Cambios en esta política

Si realizamos cambios significativos, actualizaremos la fecha en la parte superior de esta página. Los cambios que afecten al tratamiento de datos serán comunicados a través de las notas de actualización de la extensión en el Chrome Web Store.
